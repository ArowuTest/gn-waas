System Architecture
7 Microservices, 1 Unified Platform

Built on Go, PostgreSQL with TimescaleDB, and React — deployed on Render (backend) and Vercel (frontends).

🔀

API Gateway

Single entry point. JWT auth, RBAC middleware, rate limiting, request routing to all internal services.

Go · Fiber · Port 3000
🔍

Sentinel

IWA/AWWA M36 water balance engine. Detects billing anomalies, night-flow theft, and district-level NRW.

Go · TimescaleDB · Port 3002
💰

Tariff Engine

2026 PURC tiered pricing calculator. Generates shadow bills for every account to compare against GWL actuals.

Go · Port 3003
🏛️

GRA Bridge

Submits validated audit reports to GRA's VSDC API. Returns QR-signed receipts for immutable evidence.

Go · GRA VSDC API · Port 3004
📡

CDC Ingestor

Mirrors GWL's billing database via Change Data Capture. Keeps shadow ledger in sync without write access.

Go · PostgreSQL CDC
📷

Go · Tesseract · MinIO

Key Capabilities
Built for Ghana's Water Sector

Every feature maps directly to a requirement in the GN-WAAS Master Technical Specification v3.0.

🔐

12-Role RBAC + Row-Level Security

Keycloak-backed authentication with PostgreSQL RLS on every district-scoped table. No cross-district data leakage possible at the database layer.

⚖️

Shadow Billing Reconciliation

Independent Lago-style ledger calculates bills using 2026 PURC tariffs. Flags any account where GWL's bill deviates >15% from the shadow bill.

🌊

IWA/AWWA M36 Water Balance

Full water balance per district: System Input Volume, Authorised Consumption, Real Losses, Apparent Losses, NRW percentage — updated nightly.

🚨

Night-Flow Theft Detection

2–4 AM bulk meter analysis separates physical leaks (constant flow) from commercial theft (pulsing flow). Triggers automatic field job dispatch.

📍

GPS-Locked Field Verification

Mobile app requires GPS proximity to meter location before photo capture. Device ID, GPS precision, and photo hash stored as immutable evidence.

🧾

GRA E-VAT Integration

Every confirmed audit event is submitted to GRA's VSDC API. Returns a QR-signed receipt that locks the audit record permanently.

🗄️

28 Idempotent Migrations

All migrations use IF NOT EXISTS guards and DO blocks for constraint creation. Safe to re-run on any environment without data loss.

📊

TimescaleDB Hypertables

meter_readings, officer_gps_tracks, and revenue_recovery_events are TimescaleDB hypertables for efficient time-series queries at scale.

🏗️

NITA Sovereign Deployment

Designed for Ghana's National Information Technology Agency infrastructure. No foreign cloud dependency for core data processing.
